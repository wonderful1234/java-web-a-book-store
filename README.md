# java-web-a-book-store
java web 在线书城系统
<h3>本项目采用DAO+MVC模式。基于eclipse开发</br>
数据库方面采用postgresql+redis。</br>
postgresql用来存储用户的信息和书籍信息和管理员信息。</br>
redis用来存储购物车信息和用户的短信验证码。</br>
数据库连接池采用c3p0。</br>
短信发送采用阿里云短信发送API。</br>
注册页面采用ajax异步提交。</h3>
下面是war文件，可直接导入到eclipse。</br>
